How to get a ESP32 to connect with Linux.

With your ESP32 not connected, Run the following command.

# ls /dev/tty*

Now Connect your ESP32 with your computer using a mini usb cable and repeat the previous command

# ls /dev/tty*

The new device is your ESP32. in my case it is /dev/ttyUSB0.

Now we need to add our user in the usergroup that got access to this device. To find that group we do:

# stat /dev/ttyUSB0

In the reply we see in my case Gid:( 14/ uucp) meaning uucp is the usergroup we looking for.

# sudo usermod -a -G uucp $USER

Logout and back in to make the changes have effect.

Run Putty. For Connection type choose Serial. As serial line use /dev/ttyUSB0. For speed we use 115200. Then press open.

Press the reset button on your ESP32. You should now see the communication of the ESP32 in Putty.
